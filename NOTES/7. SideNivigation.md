# Side Navigation (Scroll-Spy) Implementation

Add a floating dot-based navigation on the right side of the screen that highlights the active section as the user scrolls and allows quick jumping to specific sections.

## User Review Required

> [!IMPORTANT]
> The side navigation is set to be hidden on mobile devices (width < 1024px) to avoid cluttering the UI. It will be visible on tablets and desktops.

## Proposed Changes

### [New Component] SideNav

#### [NEW] [SideNav.jsx](file:///Users/shooting/GitHub/Vibe-Coding-Personal-Website/src/components/SideNav.jsx)
- Create a list of navigation items: Home (Hero), Experience, Education, Media, Projects, Publications.
- Use `IntersectionObserver` to detect the currently active section based on the user's scroll position.
- Implement `scrollToSection` using `scrollIntoView({ behavior: 'smooth' })`.

#### [NEW] [SideNav.css](file:///Users/shooting/GitHub/Vibe-Coding-Personal-Website/src/components/SideNav.css)
- Fixed position on the right side.
- Glassmorphism style labels that appear on hover.

- Responsive design: hidden on small screens.
